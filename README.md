# dungeonGenerator

Implementation of  Wave Function Collapse Algorithm (generation of a 2D map with a define rule file containing tileset)

## dependencies
- gtkmm

## possibilities
- choose the rule file
- render map
- save map
- choose map size
- move on a map bigger than 20x20
