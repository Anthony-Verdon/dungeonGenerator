# dungeonGenerator

implémentation du Wave Function Collapse Algorithm (génération d'une carte 2D)

dépendances:
- gtkmm

options disponibles:
- choix du rule file
- affichage de la map
- sauvegarde de la map
  
options à rajouter:
- choix de la taille de la map (bloqué à 20x20)
- pouvoir se balader dans une map supérieur à du 20x20
